

## CRM-система для декларантов и сертификатчиков

### Обзор проекта
Полнофункциональная CRM-система с тёмным современным дизайном для управления декларациями, сертификатами, задачами и сотрудниками. Поддержка трёх языков: русский, узбекский и английский.

---

### Этап 1: Базовая архитектура и авторизация

**1.1 Система авторизации**
- Страница регистрации с полями: ФИО, email, телефон, выбор деятельности (Декларант/Сертификация), пароль
- Страница входа (email + пароль) с кнопкой "Забыли пароль?"
- Модальное окно регистрации/входа в фирму (название + ИНН из 9 цифр)
- Автоматическое определение фирмы по ИНН при входе сотрудника

**1.2 Система ролей**
- Админ (отдельная панель управления)
- Директор (назначается админом)
- Старший (назначается админом)
- Сотрудники

**1.3 Мультиязычность**
- Переключатель языков в интерфейсе
- Файлы локализации для RU/UZ/EN

---

### Этап 2: Навигация и лейауты

**2.1 Боковое меню для декларантов**
- Аватар, имя и телефон пользователя
- Разделы: Дашборд, Декларации, Сертификаты, Задачи, Документы, Клиенты, Сотрудничающие компании
- Дополнительно для директора/старшего: Мои сотрудники, Запросы
- Настройки и Выход

**2.2 Боковое меню для сертификатчиков**
- Аналогичная структура без раздела "Декларации"

**2.3 Боковое меню для админа**
- Дашборд, Фирмы, Пользователи, Запросы, Выход

**2.4 Верхняя панель**
- Кнопка уведомлений с индикатором
- Кнопка звука (вкл/выкл)
- Переключатель языка

---

### Этап 3: Дашборды

**3.1 Дашборд декларантов**
- Карточки статистики: активные/выполненные/просроченные задачи
- Статистика декларации и сертификатов
- Списки последних 10 декларации и сертификатов
- Фильтр по периоду

**3.2 Дашборд сертификатчиков**
- Карточки статистики задач и сертификатов
- Списки активных и просроченных сертификатов

**3.3 Дашборд директора**
- Общая статистика + возможность выбрать конкретного сотрудника

**3.4 Дашборд админа**
- Количество компаний и пользователей
- Линейный график роста
- Активные запросы

---

### Этап 4: Раздел Декларации

**4.1 Форма добавления декларации**
- Номер поста (5 цифр), дата, номер декларации (7 цифр) → формат 26001/22.12.2025/0010722
- Выбор клиента с поиском
- Режим декларации (29 вариантов)
- Номер машины и тип (8 вариантов), можно добавить несколько
- Примечание и прикрепление файлов/папок из документов

**4.2 Управление декларациями**
- Редактирование, удаление, перенаправление сотруднику
- Группировка в папки
- Фильтры: Мои/Все/По сотруднику
- Поиск по всем полям

**4.3 Детальная страница декларации**
- Все данные + связанные сертификаты и задачи

---

### Этап 5: Раздел Сертификаты

**5.1 Для декларантов - создание заявки**
- Выбор фирмы сертификатчиков или "Для себя"
- Тип, срок, номер сертификата (или "Заполнит сертификатчик")
- Выбор клиента, привязка декларации
- Примечание, файлы, автоматическая дата

**5.2 Для сертификатчиков - обработка заявок**
- Статусы: Завершено, В процессе, Ждём оплату, На проверке, Отклонить
- Механизм подтверждения документов
- Прикрепление файлов для проверки
- История всех действий

**5.3 Управление и поиск**
- Распределение директором между сотрудниками
- Фильтрация и поиск

---

### Этап 6: Раздел Задачи

**6.1 Создание задачи**
- Выбор фирмы и сотрудника
- Название, примечание
- Приоритет: Срочный/Высокий/Обычный
- Статус: Новая/В работе/Ожидание/На проверке/Завершена/Отменена/Заморожена
- Дедлайн (дни или дата)
- Прикрепление файлов, декларации, сертификатов

**6.2 Управление задачами**
- Редактирование, удаление
- История изменений статусов
- Директор видит все задачи, сотрудники только свои

---

### Этап 7: Раздел Документы

**7.1 Файловый менеджер**
- Загрузка файлов любого формата
- Создание папок
- Привязка к клиентам
- Настройки доступа: личный/общий/для выбранных сотрудников

**7.2 Фильтрация**
- По сотрудникам
- По клиентам

---

### Этап 8: Раздел Клиенты

**8.1 Добавление клиента**
- Название фирмы, ИНН, имя директора
- Настройки доступа
- Примечание

**8.2 Карточка клиента**
- Все данные + связанные декларации, сертификаты, документы, задачи

---

### Этап 9: Сотрудничающие компании

**9.1 Запрос на сотрудничество**
- Поиск по ИНН, автозаполнение названия
- Отправка запроса

**9.2 Управление партнёрами**
- Список партнёров
- Возможность удаления

---

### Этап 10: Управление сотрудниками

**10.1 Список сотрудников**
- Свои коллеги + коллеги партнёрских компаний

**10.2 Действия директора**
- Удаление с перераспределением данных
- Блокировка с уведомлением

---

### Этап 11: Запросы и уведомления

**11.1 Раздел Запросы**
- Запросы на вступление в фирму
- Запросы на сотрудничество
- Действия: Принять/Отклонить

**11.2 Система уведомлений**
- Всплывающие уведомления
- Звуковые сигналы (настраиваемые)
- Счётчик непрочитанных

---

### Этап 12: Настройки профиля

- Изменение ФИО, email, телефона, пароля
- Загрузка фото профиля
- Подтверждение через код (подготовка для Telegram)

---

### Этап 13: Панель администратора

**13.1 Авторизация админа**
- Логин + пароль + код (обновляется каждые 24 часа)

**13.2 Управление фирмами**
- Список, блокировка, удаление, отправка сообщений

**13.3 Управление пользователями**
- Назначение ролей (директор, старший)
- Блокировка, удаление, сообщения
- Детальная информация о пользователе

---

### Технические детали

**API-слой**
- Абстрактный слой для REST API запросов
- Типизированные интерфейсы для всех сущностей
- Обработка ошибок и состояний загрузки

**Состояние приложения**
- React Context для авторизации и языка
- React Query для серверных данных
- Локальное хранение настроек

**Дизайн-система**
- Тёмная тема с современными акцентами
- Адаптивность для всех экранов
- Компоненты на базе shadcn/ui

---

### Результат
Полностью готовый frontend для CRM-системы, который легко подключить к вашему backend через REST API. Все компоненты будут типизированы и подготовлены для реальных данных.
